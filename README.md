Descripción general del sistema:

Este proyecto es un prototipo inicial de un sistema orientado a objetos diseñado para la agencia de turismo **Llanquihue Tour**, ubicada en la Región de Los Lagos. El objetivo principal es digitalizar y estructurar la información de las personas involucradas en la operación (clientes, guías y administradores), resolviendo problemas de duplicación de datos, falta de validación y optimizando la gestión manual mediante principios fundamentales de POO como encapsulamiento, herencia y composición.
